Подготовка ПК

Обязательные действия перед первым запуском продукта.

Пропуск шагов = повышенный шанс ошибок.
Отключите Защитник Windows и контроль в реальном времени.
Если установлен СТОРОННИЙ антивирус — удалите его (Windows Defender не удаляется).
Если Defender не отключается вручную: скачайте DControl.
Запустите DControl от имени администратора -> Disable Windows Defender -> перезагрузите ПК.
Очистите папки Temp и перезагрузите ПК.
Проверьте синхронизацию даты и времени Windows.

Куда нажимать (пошагово)

1) Windows Security -> Virus & threat protection -> Manage settings -> Real-time protection: Off.
2) Пуск -> Параметры -> Приложения -> Приложения и возможности -> удалить только СТОРОННИЙ антивирус (если установлен).
3) Win+R -> %temp% -> Ctrl+A -> Delete.
4) Параметры -> Время и язык -> Дата и время -> Включить авто-время и нажать Sync now.

Запуск продукта

Рекомендуется запускать лоадер и игру на чистой системе после перезагрузки.
Скачайте актуальный лоадер через ключ/кабинет.
Запустите лоадер от имени администратора.
Выберите продукт и нажмите «Запуск».
После успешной инициализации сразу запускайте игру.
Откройте меню горячей клавишей (INSERT / HOME).

Частые ошибки

Нажмите на ошибку, чтобы открыть решение.

#5WebView2 Runtime not found▶
  • Установите Microsoft Edge WebView2 Runtime: скачать установщик.
  • Перезагрузите ПК и повторите запуск.
#8Injection error▶
  • Удалите античиты/антивирус, очистите Temp, перезагрузите ПК.
  • Запускайте лоадер строго до запуска игры.
#12The system is under HVCI▶
  • Отключите Изоляцию ядра / Memory Integrity.
Windows Security -> Device security -> Core isolation details -> Memory integrity: Off.
#15Failed to read connection▶
  • Перезагрузите роутер; при статическом IP используйте VPN на этапе лоадера.
#26Лоадер не запускается▶
  • Откройте: Пуск -> Параметры -> Сеть и Интернет -> Изменение параметров адаптера.
  • ПКМ по вашему подключению -> Свойства -> Протокол Интернета версии 4 (TCP/IPv4).
  • Укажите DNS: Предпочитаемый 1.1.1.1, Альтернативный 1.0.0.1 -> Сохранить -> перезапустить роутер.
