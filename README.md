# GHOST Docs

Three-language documentation website for GHOST products:

- Russian (RU)
- English (EN)
- Chinese (ZH)

## Local preview

Open `index.html` in your browser.

## Deploy to GitHub Pages

1. Create a new GitHub repository (example: `ghost-docs`).
2. Upload this folder content (`index.html`, `README.md`).
3. In repository settings, open **Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main` and folder: `/ (root)`.
6. Save and wait 1-2 minutes.

Your link will be:

`https://<your-username>.github.io/<repo-name>/`

## Notes

- Replace Telegram contact in `index.html` (`@GHOST_SUPPORT`) with your real support account.
- If you want a custom domain, add a `CNAME` file and point DNS to GitHub Pages.
